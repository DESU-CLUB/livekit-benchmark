## Architecture

```
POST /webhook  ──► express.raw({ type: 'application/webhook+json' })
                        │
                        ▼
               receiver.receive(body, authHeader)   ← HMAC validation
                        │
                   event.event string
                        │
              ┌─────────┼──────────────────┐
              ▼         ▼                  ▼
        room_started  participant_joined  track_published  … (11 handlers)
              │
              ▼
        logEvent() → events.log + console
```

## Key design decisions

| Requirement | Implementation |
|---|---|
| Raw body for HMAC | `express.raw({ type: 'application/webhook+json' })` applied **per-route only** — no global `express.json()` |
| Signature validation | `WebhookReceiver.receive(body, authHeader)` — throws on invalid JWT, returns `401` |
| Buffer → string | `req.body.toString('utf8')` before passing to receiver |
| Always ACK to LiveKit | Returns `200` even if the handler throws, preventing retries |
| Unknown events | Logged with `unknown_event` type rather than silently dropped |

## Handled event types
`room_started`, `room_finished`, `participant_joined`, `participant_left`, `track_published`, `track_unpublished`, `egress_started`, `egress_updated`, `egress_ended`, `ingress_started`, `ingress_ended`
