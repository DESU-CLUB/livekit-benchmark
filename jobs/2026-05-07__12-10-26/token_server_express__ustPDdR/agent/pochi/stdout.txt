**What it does:**
- Imports `express` and `AccessToken` from `livekit-server-sdk`
- Reads `LIVEKIT_API_KEY` and `LIVEKIT_API_SECRET` from environment variables
- Listens on port `3000`
- `GET /token?room=<room>&identity=<identity>` — returns `{ "token": "<jwt>" }` with grants `roomJoin`, `canPublish`, `canSubscribe`
- Returns HTTP 400 with an error message if `room` or `identity` is missing
